# 翠巒祭 販売システム - Quick QR

[sell.suiranfes.blue](https://sell.suiranfes.blue)

## 技術

- Vite + React (TypeScript)
- データ保存: Google Spread Sheet
- デザイン: Material UI
- フォント: IBM Plex Sans, IBM Plex Sans JP

### 必要環境

- `git`
- `node.js`
- `npm`

### 動作方法

1. `npm i`
1. `npm start`
1. ブラウザで `http://localhost:3000` にアクセス
